This project from Frontend Mentor was build using Bootstrap 5 for the layout and custom css for specific design requirements.
The primary challenge involved reproducing the unique and complex visual style, which required significant time to impolement
non-standard elements like the clipped corner, progressive bar, and adaptability for mobile devices. To navigate these styling hurdles,
I utilized official documentaion for Bootstrap 5 and AI assistance to research methods for handing gradients, positioning, and custom shapes.
This project was a great challenge for me. Implementing this unique design took a lot of time, but it provided me with valuable expirience that
will definitely be useful in my future works.
